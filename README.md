# Brandon Theard — Portfolio (GitHub Pages)

This repo powers the live portfolio site hosted on **GitHub Pages**.

## What this is
A one-page React portfolio with:
- About section
- 4 project cards (3 live apps + EarningsEdge repo)
- Contact links (GitHub / LinkedIn / email)

## Update your contact info
Edit:
- `src/App.tsx` → `CONTACT.linkedin` and `CONTACT.email`

## Deploy
This repo deploys automatically on every push to `main` via GitHub Actions.

In GitHub:
1. **Settings → Pages**
2. Under **Build and deployment**, select **GitHub Actions**

Then push to `main` and you’re live at:
- https://btheard3.github.io/
